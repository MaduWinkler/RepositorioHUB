# PortfolioHUB

Este repositório foi desenvolvido com o objetivo de reunir projetos acadêmicos e pessoais desenvolvidos durante os estudos na área de computação.

O projeto utiliza práticas de versionamento com Git e GitHub, além de organização de código e documentação.

## Estrutura do Repositório

📁 projetos-academicos  
Projetos desenvolvidos durante atividades da graduação.

📁 projetos-pessoais  
Projetos desenvolvidos para estudo e prática.

📁 documentacao  
Arquivos explicativos e documentação do projeto.

## Tecnologias utilizadas

- Git
- GitHub
- HTML
- CSS
